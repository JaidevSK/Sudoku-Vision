# Sudoku-Vision

The Sudoku Vision app is a sophisticated tool designed to facilitate Sudoku resolution directly from images. This innovative application leverages Python and Streamlit to empower users to effortlessly tackle Sudoku challenges. Users can input digits manually, upload text files, or even capture images of Sudoku puzzles for quick resolution. By utilizing advanced image processing techniques, the app can accurately extract Sudoku grids from images and solve them efficiently. Sudoku Vision offers a seamless and user-friendly experience, making Sudoku solving more accessible and engaging for enthusiasts of all skill levels.

## Install the Application:
Clone the Repository:
```
git clone https://github.com/JaidevSK/Sudoku-Vision.git
```
Install the Requirements:
```
pip install -r requirements.txt
```
